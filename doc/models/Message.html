Overview

Model to handle message created by dispute between admins and users, generally associated to an parking session.

Table's Columns

  • text => [text] Message body
  • author => [string] Sender of the message can be an admin or a user
  • read => [boolean] If the message was read
  • to => [string] Receiver of the message can be an admin or a user
  • template => [integer] Reason of the message
  • title => [string] Title automatically set depending on the template
  • parking_session_id => [bigint] ID reference to a parking session
  • created_at => [datetime]
